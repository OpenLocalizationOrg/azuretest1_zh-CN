#### 若要安装管理门户通过定期更新

1. 在**设备**页中，选择您要在其安装更新的设备。

2. 导航到**设备** > **维护**和向下的滚动到**软件更新**。

3. 要检查更新，请单击**检查更新**页面底部。

4. 如果有可用的软件更新，您将看到一条消息。 单击要开始更新该设备的**安装更新**。

    在成功安装更新时通知您。

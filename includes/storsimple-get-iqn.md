#### 若要获取 Windows 主机的 IQN

1. 在 Windows 主机上启动 Microsoft iSCSI 发起程序。

2. 在**iSCSI 发起程序属性**窗口上的**配置**选项卡上，选择并复制**启动器名称**字段中的字符串。
 
    ![iSCSI 发起程序属性](./media/storsimple-get-iqn/HCS_iSCSIInitiatorPropertiesFigureIQN-include.png)

3. 保存此字符串。

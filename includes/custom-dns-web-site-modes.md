设置自定义域名功能仅适用于**共享**服务、**基本**服务和 Web 应用程序的**标准**模式。 再切换 web 应用程序从**自由**模式为**共享**、**基本**或**标准**的模式，必须首先删除支出上限就地为您 Web 应用程序的订阅。 

包括如何更改您的网站的模式，请参阅[如何扩展 web 应用程序](../article/app-service-web/web-sites-scale.md)的应用程序服务计划模式的详细信息。

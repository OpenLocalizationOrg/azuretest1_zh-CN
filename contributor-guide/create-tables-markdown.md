#在减价中创建表

减价中创建表的最简单方法是使用管道和线路。

 ![][1]

您可以调整用冒号的列︰

  	|-----:| - this is right aligned
  	|:-----| -  this is left aligned
  	|:-----:| - this is centered

如果您使用 HTML 表格，您减价不呈现两个表之间，您需要关闭 TABLE 标签后添加结束 BR 标签。

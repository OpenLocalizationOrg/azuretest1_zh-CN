#支持的浏览器和设备
所有现代的台式机和 tablet 设备和浏览器上运行 Microsoft Azure 门户。

### 支持的设备
门户将现代 Pc、 Mac 和平板电脑上运行很好。 目前不支持移动电话。

### 支持的浏览器
支持以下浏览器最新版本︰

- 边缘 （最新）
- Internet Explorer （11 和向上）
- Safari （7 和向上）
- 铬 （最新）
- Firefox （最新）

*不支持 safari 6 或更低。 如果您使用的 OS X，可以使用 Chrome，Firefox，或升级到 OS X Mavericks 获取 Safari 7。*

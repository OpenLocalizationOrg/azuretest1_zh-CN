# 如何查看和管理您的应用程序有关的数据的图表
用户界面的仪表板部分允许客户创建自定义的图表，以最大限度地通过提供他们想要的而不是在分析部分中搜索的准确信息的时间。 仪表板显示应用程序和在某一给定时间您应用程序的不同版本的活动用户计数的趋势 （小时/天/周/月/自定义）。 最近 7 天都按所示的趋势。

### 请参见
-  [故障排除指南的分析][链接 21]

## 仪表板
- 通过您想在仪表板上看到的用户界面的分析部分中所有图表的齿轮菜单中选择"发送到仪表板"操作，则可以对空的仪表板中添加图表。
 
## + （添加仪表板）
- "+"功能区菜单项可用于创建新的仪表板将显示功能区菜单上。
 
## 设备菜单
- 您可以使用仪表板的齿轮菜单删除、 重命名共享，或您的仪表板的计划 （电子邮件接收）。 计划选项可用于定义日程安排，您希望通过电子邮件接收您自定义的仪表板。 您可以安排每天、 每周或每月接收它。 您可以选择的和当天的一周，或者当您想要接收电子邮件每月的某一天的时间。 您的电子邮件地址和时区基于用户界面的我的帐户部分中配置的设置。

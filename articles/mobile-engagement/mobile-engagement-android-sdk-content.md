#SDK 内容

本文档列出并描述 SDK 归档文件的内容。

`mobile-engagement-VERSION.jar`

> 接洽 SDK 类文件。

`res` 文件夹

> 接洽到达 SDK 资源 (可绘制和布局文件要包括在您的应用程序中)。

`src` 文件夹

> 接洽 SDK 的源代码。
>
> 请注意服务的来源不是包括 (`com.microsoft.azure.engagement.service.EngagementService`及其相关的源代码文件)。

`documentation.html`

> 指向 SDK 下载版本相对应的文档站点的快捷方式。

其他文件是一份软件许可证。

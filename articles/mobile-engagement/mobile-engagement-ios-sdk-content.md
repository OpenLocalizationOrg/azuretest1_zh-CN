#SDK 内容

本文档列出并描述 SDK 归档文件的内容。

##接洽 SDK

`EngagementSDK`

> 核心服务 SDK 包含标题、 库和作为源代码来启用/禁用 IDFA 集合提供一个附加类。

##合作范围 SDK

`EngagementReach`

> 接洽到达 SDK 包含标题、 库和相关的资源。

##附加文件

`doc/index.html` (在`doc`文件夹)

> 服务 SDK，到达 SDK 目标 C 文档内容。

`src` 文件夹

> 服务视图控制器，并达到 SDK （针对那些想要适应交往类到其需要的超级用户） 的源代码。

# Azure 多因素身份验证中的 Azure 门户创建应用程序密码

如果在 Azure 使用多因素身份验证，想要创建通过 Azure 的门户应用程序密码。

## 在 Azure 的门户网站中创建应用程序密码

1. 登录到 Azure 管理门户
3. 在顶部，右键单击您的用户名并选择其他安全验证。
5. 在 proofup 页面顶端，选择应用程序密码
6. 单击**创建**
7. 为应用程序密码输入一个名称，然后单击**下一步**
8. 应用程序密码复制到剪贴板并将其粘贴到您的应用程序。

# 摄取在规模较大的遥测和应用程序事件
 
现代应用程序在运行时创建大量的信息。 在移动应用程序或网站上的 clickstreams 的用户操作，此遥测数据是否有价值的长期分析和实时运行状况监视或决策支持。 服务总线提供了您需要捕获和处理这些信息的工具。

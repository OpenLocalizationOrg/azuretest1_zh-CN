# 我应该选择云服务？

Azure 用于运行应用程序提供不同的宿主模型。 每个提供一组不同的服务，因此您选择哪一种取决于确切什么要做。 是云服务为您的选择吗？

| 计算选项    | 受众   |
| ------------------ | --------   |
| [应用程序服务]     | 可伸缩的 Web 应用程序、 移动应用程序、 API 的应用程序和任何设备的逻辑应用程序 |
| [云服务]   | 高度可用、 可伸缩的 n 层云应用程序的操作系统的更多控制 |
| [虚拟机] | 自定义的 Windows 和 Linux，完全控制操作系统的虚拟机 |

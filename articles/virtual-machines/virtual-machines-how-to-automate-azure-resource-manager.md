# 如何自动使用 Azure 资源管理器

Azure 的资源管理器可以简化部署和管理复杂的应用程序运行在 Azure 的基础结构服务。 许多应用程序要求的计算、 网络和存储资源，如虚拟网络、 存储帐户和虚拟机的网络接口的组合。 Azure 的资源管理器可以使用或者构建一个 JSON 模板部署和管理所有这些资源放在一起作为单个应用程序。

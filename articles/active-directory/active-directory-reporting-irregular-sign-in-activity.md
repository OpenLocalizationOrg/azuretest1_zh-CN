# 在活动中的不规则的符号

不规则号单元是指那些由我们的机器学习算法，结合位置和设备中的一个反常号"根本不可能旅行"条件的基础。 这可能表明，黑客已经成功地注册中使用此帐户。
如果我们遇到了在 30 天或更少的范围内的事件中的 10 个或更多反常号，我们将向全局管理员发送电子邮件通知。 请务必在您的安全发件人列表中包含 aad-alerts-noreply@mail.windowsazure.com。

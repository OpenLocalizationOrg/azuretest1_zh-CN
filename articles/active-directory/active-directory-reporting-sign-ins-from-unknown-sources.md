# 从未知的来源的签名宏
<p>此报表指明用户成功登录到您的目录，而分配已由 Microsoft 被识别为一个匿名代理服务器的 IP 地址 (或 IP) 的 IP 地址的客户端。 想要隐藏他们的计算机 IP 地址，以及用于恶意用户通常使用这些代理。 </p><p> 此报表中的结果将显示用户成功登录到您的目录从该地址和代理服务器的 IP 地址的次数。</p>
